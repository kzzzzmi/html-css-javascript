웹 브라우저의 HTML 문서 렌더링 과정

1. 불러오기(loading)
- 로더가 서버로부터 전달 받은 리소스 스트림을 읽는 과정, 읽으면서 어떤 파일인지, 데이터인지 파일을 다운로드할 것인지 결정함

2. 파싱(parsing)
- 웹 엔진이 가지고 있는 HTML/XML 문서를 파싱해서 DOM Tree를 만듬
- 파싱 : 문서를 파싱한다는 것은 브라우저가 코드를 이해하고 사용할 수 있게 변환하는 것

3. 렌더링 트리란?
- DOM Tree는 내용을 저장하는 트리로 자바스크립트에서 접근하는 DOM 객체를 사용할 때 이용하는 것
 
4. CSS 스타일 결정
- 선택자에 따라서 적용되는 태그가 다르기 때문에 모든 css 스타일을 분석해 태그에 스타일을 적용하는 과정

5. 레이아웃
- 렌더링 트리가 생성될 때 위치나 크기를 가지고 있지 않기 때문에 객체들에게 위치와 크기를 정해주는 과정을 레이아웃이라고 함
